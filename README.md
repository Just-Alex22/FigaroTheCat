# FigaroTheCat

## What is FigaroTheCat?
**FigaroTheCat** is a telegram bot that let's you pick between __yes__ and __no__ with the help of a python-based randomizer. It can also act as a dumb companion which you can ask questions to. The only downside is that it only answers with meows, it's still fun tho.

## Usage:
- **Meow:** Yes
- **Meow Meow:** No
- **Meeeoooow:** No idea

## Example:
[User:] Should I install linux on my gaming pc?

[Figaro:] Meow

## Installation:
Just message @figarothecat_bot on telegram, no installation needed!
